# Alpha Vantage — Tool Reference Guide

**Version:** 4.0.0 · **Tools:** 35 · **API Key Required:** Yes (`alpha_vantage.api.key` in `config/application.yml`)

---

## Overview

SAJHA provides 35 Alpha Vantage tools covering stock quotes, time series, financial statements, 15 technical indicators, forex, crypto, and economic data. All tools use the Alpha Vantage REST API.

**API Key:** Get a free key at [alphavantage.co](https://www.alphavantage.co/support/#api-key).
