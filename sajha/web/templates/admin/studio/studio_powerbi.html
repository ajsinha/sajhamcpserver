Quick Tips
  • The report will be returned as base64 encoded data
  • Caller decodes base64 to get the actual PDF/PPTX/PNG file
  • Set POWERBI_CLIENT_SECRET environment variable before running
  • Ensure Service Principal has Report.Read.All permission
  • Add Service Principal to the PowerBI Workspace as Member
