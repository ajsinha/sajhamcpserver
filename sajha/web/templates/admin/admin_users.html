Page Glossary

User Management: Administrative functions for creating, editing, and deleting user accounts.

RBAC (Role-Based Access Control): Authorization approach where permissions are assigned to roles (admin, user, readonly).

User ID: Unique identifier for each user in the system.

Role: A named set of permissions (e.g., admin has full access, readonly has view-only access).

Password Hash: Encrypted representation of user password using SHA-256.

Session Token: Temporary credential issued after successful login.
