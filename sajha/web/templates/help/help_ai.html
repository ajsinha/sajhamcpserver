Composite Tools
Sibling Pattern (Parallel)

Multiple tools run concurrently, results merged. Example: market_snapshot = yahoo_quote + fred_vix + fred_fed_rate — one call, three results. Confidence: weakest-link (min).

Parent-Child Pattern (Fan-out)

Master tool runs first, child executes per record. Example: portfolio_dive = duckdb_query → fmp_profile(each ticker). Confidence: multiply (Giry bind).

Build one: MCP Studio → Composite Builder → pick master tool → add steps → preview schemas → save.

Composition Framework

Category-theory-inspired tool composition with mathematical guarantees about confidence, isolation, and error handling.

Kleisli Arrows

Every tool wrapped in StepResult envelope. Errors short-circuit the pipeline. Traces accumulate. Confidence compounds through the chain.

ParamLens

Surgical parameter projection. Child tools see ONLY mapped fields via $.field / $input.field syntax. Prevents accidental coupling.

Entropy Guard

Cumulative confidence tracking. Sequential: multiply. Parallel: min (weakest link). Refuses pipelines exceeding entropy threshold.
