Overview

SAJHA MCP Server is a comprehensive Python implementation of the Model Context Protocol (MCP) built on FastAPI. It provides a standardized interface for AI tools and services, serving 497 tools across financial markets, government data, search engines, analytics, and enterprise integrations through a single MCP-compliant endpoint.

The server supports three transports (HTTP POST, SSE, WebSocket), includes an embedded LLM gateway with 6 AI providers, semantic tool discovery, composite tool orchestration, OpenTelemetry observability, multi-tenancy, a plugin system, and a full web UI with 12 interactive screens. Version 5.0.0 is fully compliant with MCP specification 2025-11-25 — including Tasks (async tracking), Elicitation (form + URL modes), Sampling with tool calling, tool icons, and Origin validation for Streamable HTTP.

MCP Specification Compliance
Protocol Version: 2025-11-25 (latest)
Released November 25, 2025 by the MCP Steering Group
Fully Compliant
Tasks (SEP-1686)

Async tracking for long-running requests. Create, poll, cancel tasks. States: working → completed/failed/cancelled.

Elicitation (SEP-1330)

Server-initiated user input. Form mode (structured JSON Schema forms) and URL mode (redirect to OAuth/consent pages).

Sampling + Tools (SEP-1577)

Server-initiated LLM calls with tool definitions and toolChoice. Enables server-side agent loops.

Icons (SEP-973)

Icon metadata for tools, resources, and prompts. Supports URL and emoji icon types for richer UI rendering.

TRANSPORT COMPLIANCE

HTTP POST /mcp (stateless) · SSE /mcp/sse (Streamable HTTP with Origin validation, polling support) · WebSocket /mcp/ws (full-duplex). JSON-RPC 2.0 on all three. JSON Schema 2020-12 dialect.

SECURITY COMPLIANCE

Tool errors return isError:true (not protocol errors) for model self-correction. Origin header validation (HTTP 403). PKCE support. OAuth 2.1 flows. RBAC + API key + JWT + OAuth SSO.

The Meaning of SAJHA (साझा / ساجھا)
साझा
SAJHA (Hindi: साझा, Urdu: ساجھا) — "Shared" or "Common"

SAJHA is a word from Hindi and Urdu meaning "shared," "common," "mutual," or "collaborative." It embodies collective ownership, partnership, and working together. In South Asian cultures, sajha represents the spirit of community cooperation where resources and knowledge are shared for mutual benefit.

How SAJHA MCP Server embodies this spirit: True to its name, SAJHA creates a shared bridge between AI systems and the world's data. It democratizes access to 497 financial, government, and enterprise tools through a single, standardized protocol. The server enables collaborative tool creation through MCP Studio, where teams can build and share custom integrations without deep coding. By providing hot-reload configuration, role-based access, and a zero-dependency client SDK, SAJHA fosters a communal environment where knowledge and capabilities flow freely between humans and AI.
