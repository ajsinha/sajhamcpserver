Overview

SAJHA MCP Server is a comprehensive Python-based implementation of the Model Context Protocol (MCP) that provides a standardized interface for AI tools and services. Built for production environments, it offers robust security, scalability, and extensibility.

The server supports both HTTP REST APIs and WebSocket connections for real-time bidirectional communication, making it suitable for a wide range of AI integration scenarios.

Core Capabilities
Full MCP Compliance

Complete implementation of the Model Context Protocol specification with JSON-RPC 2.0 support.

Dual Transport

Supports both HTTP REST API and WebSocket connections for flexible integration options.

Hot-Reload System

SAJHA MCP Server features a comprehensive hot-reload system that automatically detects and applies configuration changes without requiring a server restart. This enables zero-downtime updates to tools, users, API keys, and prompts.

What Gets Hot-Reloaded
Component Files Interval
Users config/users.json 5 min
API Keys config/apikeys.json 5 min
Tool Configs config/tools/*.json 5 sec
Tool Modules tools/impl/*.py 5 sec
Prompts config/prompts/*.json 5 min
How It Works
  1. Add a tool: Create a new JSON config in config/tools/ and Python module in tools/impl/ - appears within 5 seconds
  2. Modify a tool: Edit the JSON or Python file - changes apply within 5 seconds
  3. Delete a tool: Remove the JSON file - disappears from system and docs within 5 seconds
  4. Update users/API keys: Edit JSON files - changes apply within 5 minutes
Force Reload: Use POST /api/admin/hot-reload/force to trigger immediate reload of all configurations.

Key Strengths
  • Financial Data Focus: Unmatched coverage of central banks, SEC filings, and market data tools
  • Enterprise Security: Multi-layer authentication with API keys, sessions, and granular permissions
  • Production Ready: Web UI, monitoring, logging, and admin features out of the box
  • Self-Documenting: Live markdown viewer with architecture diagrams for all 150+ tools
  • Comprehensive Hot-Reload: Auto-reload within 5 minutes for users.json, apikeys.json, tool configs, Python modules, and prompts - zero downtime changes
  • Flexible Access: Both session tokens and API keys supported with regex-based tool permissions
  • MCP Studio: Visual Tool Creation Platform with three methods: Python Code Creator, REST Service Tool Creator, and DB Query Tool Creator
  • Dark Theme Support: Comprehensive light/dark mode with 3,200+ lines of CSS ensuring accessibility across all pages
Considerations
  • Python-Based: Requires Python 3.9+ environment (not embedded like Claude Desktop)
  • Self-Hosted: You manage the server infrastructure (can be pro or con)
  • API Keys Required: Some external tools (Google, Tavily) need their own API keys
  • Learning Curve: Rich features require some initial setup and configuration
  • Resource Usage: 150+ tools use more memory than minimal implementations
Bottom Line: SAJHA MCP Server is ideal for organizations that need a production-ready, enterprise-grade MCP implementation with comprehensive financial data tools, robust security, and a full-featured admin interface. It's particularly well-suited for fintech, research, and enterprise AI integration projects where security and data access control are paramount.

Page Glossary

SAJHA: The name of this MCP server implementation. Derived from Sanskrit, meaning "shared" or "collaborative."

MCP (Model Context Protocol): A standardized protocol enabling AI systems to interact with external tools.

JSON-RPC 2.0: The protocol used for MCP communication, using JSON for data encoding.

Flask: A lightweight Python web framework used as the foundation for SAJHA's web layer.

WebSocket: A protocol for full-duplex real-time communication used by SAJHA.

Production-Ready: Software suitable for deployment in live environments serving real users.
