    Page Glossary

    Tool: A function exposed via MCP that AI systems can call. Each tool has a name, description, input schema, and execution logic.

    Input Schema: A JSON Schema definition describing the valid parameters a tool accepts.

    JSON-RPC: The protocol used for MCP tool calls, using JSON for request/response encoding.

    Tool Registry: The singleton component managing all registered tools, handling loading and hot-reload.

    Execute: Running a tool with provided arguments to produce results.

    Category: A grouping mechanism for organizing related tools (e.g., Financial, Search, Government).
